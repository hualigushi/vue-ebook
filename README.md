# vue-ebook

### 用到的npm包

web-storage-cache 

localstorage操作，，防止页面刷新后数据丢失。将传入的对象变成JSON进行存储，读取的时候再转为对象

vue-i18n 

国际化

vue-create-api

自定义的通用组件，以全局方式调用
